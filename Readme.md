# ImGui-Raylib Application : Template

This package is a template to make a C/C++ application using raylib and Dear ImGui.

## Features

The template comes with the following libraries as submodules:
 
 * imgui
 * imgui-filebrowser
 * implot
 * rlImGUI
  
**Raylib needs to be installed in your system to be able to compile** (You can also add raylib as a dependency and compilte it).

## Compiling

To compile and run the aplication execute the following comands

```sh
mkdir build
cd build 
cmake ..
make -j4
```
